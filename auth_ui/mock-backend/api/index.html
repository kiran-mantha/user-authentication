Mock Backend API

This is a mock backend for the Angular Authentication UI demo.

The actual API responses are simulated in the DemoInterceptor.

Available Endpoints

GET /api/users/ - List users
POST /api/users/ - Create user
GET /api/users/{id}/ - Get user details
PATCH /api/users/{id}/ - Update user
DELETE /api/users/{id}/ - Delete user
POST /api/login/ - User login
POST /api/logout/ - User logout
GET /api/user/ - Current user profile
GET /api/roles/ - List roles
GET /api/permissions/ - List permissions

Features

  • JWT Authentication with automatic token refresh
  • Role-based access control
  • User management (CRUD operations)
  • Role management with permission assignment
  • Permission management for API endpoints
  • Responsive design for mobile and desktop
  • Modern UI with loading states and error handling
